Welcome to Bitcoin

What is Bitcoin?

Bitcoin is a decentralized digital currency, without a central bank or single administrator, that can be sent from user to user on the peer-to-peer bitcoin network without the need for intermediaries.

How to Use Bitcoin

Bitcoins are transferred via a peer-to-peer network, and can be used for a variety of transactions, including buying goods and services, or trading them like stocks or currency.

Setting Up a Bitcoin Wallet

To use Bitcoin, you need to set up a digital wallet. A Bitcoin wallet allows you to receive, store, and send bitcoins.
